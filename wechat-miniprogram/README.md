# 青年周刊 - 微信小程序

《青年周刊》的微信小程序版本。

## 功能

- 📱 展示最新周刊列表
- 📖 文章阅读页面
- 🔍 分类浏览
- ↗️ 分享功能

## 开发

### 环境要求

- 微信开发者工具
- 微信小程序账号

### 本地开发

1. 使用微信开发者工具打开 `wechat-miniprogram` 目录
2. 填写自己的 AppID（或使用测试号）
3. 开始开发

### 目录结构

```
wechat-miniprogram/
├── pages/           # 页面
│   ├── index/      # 首页
│   ├── list/       # 周刊列表
│   └── detail/     # 详情页
├── images/         # 图片资源
├── app.js          # 应用逻辑
├── app.json        # 应用配置
├── app.wxss        # 全局样式
└── README.md       # 本文件
```

### 数据接口

小程序需要从服务器获取数据，可以在 `app.js` 中配置 `apiBaseUrl`：

```javascript
globalData: {
  apiBaseUrl: 'https://your-api-domain.com'
}
```

## 发布

1. 在微信开发者工具中点击"上传"
2. 登录微信公众平台
3. 提交审核
4. 审核通过后发布

## 注意事项

- 确保服务器域名已在微信公众平台配置
- 遵循微信小程序的设计规范
- 注意用户隐私保护
