## 内容板块

每期周刊包含以下板块：

1. **刊首语** - 本周话题或编辑寄语
2. **科技新势力** - AI、编程、工具推荐
3. **二次元次元壁** - ACG资讯、动漫评论
4. **游戏研究所** - 游戏评测、攻略心得
5. **青春故事会** - 成长故事、职场指南
6. **好工具** - 生产力工具推荐
7. **在看什么** - 影视、书籍、播客推荐
8. **一周图鉴** - 视觉内容精选
9. **谁在招人** - 招聘信息

## 投稿指南

欢迎投稿！请参考 [投稿指南](../CONTRIBUTING.md)。

### 投稿方式

1. 通过 GitHub Issue 提交
2. 提交 Pull Request
3. 发送邮件至：youth-weekly@example.com

### 内容要求

- 原创内容优先
- Markdown 格式
- 配图需注明来源
- 字数建议 500-3000 字
